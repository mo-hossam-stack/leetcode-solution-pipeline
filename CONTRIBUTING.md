# Contributing to LeetCode AI Formatter

Thank you for your interest in contributing!

## Ways to Contribute

- Report bugs
- Suggest new features
- Improve AI prompts for better metadata inference
- Add language support
- Improve Arabic translations
- Submit pull requests

## Development Setup

1. Fork the repository
2. Clone your fork
3. Create a feature branch

```bash
git checkout -b feature/your-feature-name
```

4. Make your changes
5. Test locally with Docker Compose
6. Submit a pull request

## Code Style

- Python: Follow PEP 8, use Black for formatting
- JavaScript: Use ESLint + Prettier defaults
- Commit messages: Use clear, descriptive messages

## Pull Request Guidelines

1. Describe what the PR does
2. Include screenshots for UI changes
3. Test your changes
4. Keep PRs focused and minimal

## Issues

- Use clear titles
- Provide reproduction steps
- Include environment details
